## About this project

This repository contains a practical Markdown guide available in Spanish and English.

The guide includes examples of:

- Headings and text formatting.
- Lists and links.
- Images and tables.
- Code blocks.
- Quotes and comments.
- Markdown editing in Visual Studio Code.
- Automatic tables of contents.

---

## Sobre este proyecto

Este repositorio contiene una guía básica de Markdown disponible en inglés y español.

La guía incluye:

- Títulos y formato de texto.
- Párrafos y listas.
- Enlaces e imágenes.
- Bloques de código y citas.
- Tablas y comentarios.
- Herramientas de Markdown en Visual Studio Code.
- Generación automática de índices.
